= Container Image Scanning and SBOMs

This lab teaches how to analyze the security posture of container images using
vulnerability scanning and Software Bill of Materials (SBOM) generation.

== Lab Overview

Participants compare Red Hat Hardened Images against traditional UBI-based images
using industry-standard security tools, and generate machine-readable SBOMs for
compliance requirements.

== Modules

* Module 01: Vulnerability Scanning with Grype — scan for CVEs, compare image security
* Module 02: SBOM Generation with Syft — create SPDX-JSON SBOMs for compliance

== Duration

12-15 minutes
